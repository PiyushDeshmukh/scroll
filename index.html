Introducing Programming



Hello there! This website aims to be a resource for autodidacts to learn Programming. We are focusing on a minimal theory followed by implementations of important algorithms and data structures. We are using Python and C++ for implementing these concepts, however I should make a point clear that the site by no means attempts to teach you programming language, but rather programming. This website is designed for those people who are already familiar with any other language and are looking forward to implement the same concepts in C++ or Python. The website can also act as a material for revision for those who are already comfortable with covered topics.

The content on the site is public. You can fork us on Github. If you have any queries or problem with the content, raise issues on the repo, or being more friendly, don't hesitate to send a merge request.
