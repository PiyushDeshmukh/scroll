Getting Started


Okay, you are ready to learn programming, that's great. But before moving ahead, we need to decide upon a programming environment. What I recommend is Ubuntu (or any Debian-based shall do). Linux is a general recommendation for learning programming. And so, the code snippets mentioned in the tutorial are implemented on Linux (to be more specific Ubuntu 16.04 on a 64-bit machine).


Code Editor


Not surprizingly, but this is the most flexible choice for you. You can use any text editor you feel comfortable working with. I generally woould recommend Atom, Sublime Text Editor. Vi or Emacs would be the best option, but if you focused on learning programming and not a bunch of other things together, you should probably not go for them.

Just a note, make sure text editor is configured to replace tabs with spaces, especially if you are going to use python as your primary language in this tutorial.

We shall be using terminal to run commands. We expect you to have a brief knowledge of Linux or atleast BASH, however, it isn't mandatory, we will provide necessary instructions wherever we feel appropriate.


C++


The code snippets here are written in gcc/g++ 5.3.1. Check your C++ version before proceeding. You may use the following command.

g++ --version


Python


We shall be using python3.5 to implement all the algorithms and the data structures. To check the version on your system, try 

python3 --version
