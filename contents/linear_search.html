Linear Search



Linear Search is arguably one of the simplest concepts to start with. Linear search deals with detecting the presence of a particular element in a list. In addition, it can return the index in the list where that element was found.

This searching approach as the name suggests, searches in a sequential fashion, i.e. it compares one element of the list to the corresponding search item and in case of fallacy, it then proceeds to the next element in the list and so on until it reaches the end of the list.
