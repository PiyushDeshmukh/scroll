Linear Search



Linear Search is arguably one of the simplest concepts to start with. Linear search deals with detecting the presence of a particular element in a list. In addition, it can return the index in the list where that element was found.

This searching approach as the name suggests, searches in a sequential fashion, i.e. it compares one element of the list to the corresponding search item and in case of fallacy, it then proceeds to the next element in the list and so on until it reaches the end of the list.

This works by maintaining an index variable, which stores the current index of the list which is being examined. We start from the zero'th index (in most programming languages including C++ and Python) and then proceed towards higher indices. At each index, we check whether the value of the element stored at that index is equal to the search item, in case if it is equal, we return the value of the index, else we proceed to the next index by incrementing the index variable, and repeat the same until we reach the end of the list. If the search item is not found, we return a pre-determined value (say -1), indicating that the element was not found in the list.
